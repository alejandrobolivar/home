Objetivo del Curso

Objetivo General: Aprender a diseñar, crear y manipular bases de datos relacionales, dominando el lenguaje estándar SQL para la gestión de información.


Las bases de datos son el corazón de cualquier aplicación moderna. En este curso usamos SQLite por su simplicidad y potencia, pero todo lo que aprenderás (SQL estándar) es aplicable directamente a sistemas gigantes como PostgreSQL, MySQL o SQL Server. Pasarás de guardar datos en archivos desordenados a estructuras relacionales robustas.

¿Para quién es este curso?

Mobile & Web Devs

SQLite es el estándar nativo en Android y iOS, y esencial para el desarrollo web backend (Django, Flask).

Analistas de Datos

Si quieres dejar de depender de Excel y realizar consultas complejas en segundos, SQL es la herramienta definitiva.

Contenido Programático

1. Diseño de Base de Datos

  • Teoría de Bases de Datos Relacionales (RDBMS).
  • Modelo Entidad-Relación: Diagramado.
  • Tipos de datos en SQLite.
  • Normalización (1FN, 2FN, 3FN) para evitar redundancia.

2. Fundamentos SQL

  • DDL: CREATE TABLE, DROP.
  • Manipulación (CRUD):
  • INSERT INTO (Crear).
  • UPDATE (Actualizar).
  • DELETE (Borrar).

3. Consultas Avanzadas

  • Consultas básicas: SELECT * FROM.
  • Filtrado de datos: WHERE, LIKE.
  • Ordenamiento: ORDER BY.
  • Agregaciones: COUNT, SUM, AVG.

4. Relaciones (JOINs)

  • Claves Primarias (PK) y Foráneas (FK).
  • Combinar tablas con INNER JOIN.
  • LEFT JOIN para datos faltantes.
  • Subconsultas y vistas básicas.
