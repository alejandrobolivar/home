Objetivo del Curso

Objetivo General: Optimizar el flujo de trabajo en el diseño asistido por computadora mediante el uso de bloques, referencias externas y preparación profesional de planos para impresión.


En el nivel básico aprendiste a dibujar; en este nivel aprenderás a producir. Este curso está diseñado para que dejes de perder tiempo editando elementos uno por uno. Dominarás la gestión de bases de datos del dibujo (bloques con atributos, tablas) y entenderás definitivamente cómo funciona el Espacio Papel (Layouts) para que tus planos salgan impresos a la escala exacta, siempre.

¿Para quién es este curso?

Dibujantes Activos

Para quienes ya usan AutoCAD pero sienten que sus procesos son lentos y quieren automatizar tareas repetitivas.

Coordinadores CAD

Para quienes necesitan estandarizar estilos de impresión, capas y formatos de presentación en su oficina.

Contenido Programático

1. Bloques Avanzados

  • Creación y Redefinición de Bloques BLOCK.
  • Editor de Bloques y parámetros base.
  • Bloques con Atributos (Rótulos inteligentes) ATTDEF.
  • Extracción de datos a tablas.

2. Gestión Externa

  • Referencias Externas XREF: Trabajo colaborativo.
  • Superposición vs. Enlace (Overlay vs Attach).
  • Gestión de imágenes y PDFs importados.
  • Creación y estilo de Tablas TABLE.

3. Acotado y Estilos

  • Administrador de estilos de cota DIMSTYLE.
  • Cotas anotativas (ajuste automático de escala).
  • Directrices múltiples MLEADER para notas.
  • Campos de texto automáticos.

4. Salida de Planos

  • Diferencia Model Space vs Paper Space LAYOUT.
  • Ventanas gráficas MVIEW y escalas (1:50, 1:100).
  • Configuración de plumillas (CTB/STB).
  • Impresión a PDF y Ploteo PLOT.
