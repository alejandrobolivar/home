Objetivo del Curso

Objetivo General: Dominar la librería Pandas de Python para la carga, limpieza, manipulación y análisis exploratorio de grandes volúmenes de datos.


En el mundo real, los datos nunca vienen limpios. Este curso te enseñará a enfrentarte al "Data Wrangling": tomar datos desordenados, corregir errores, cruzar múltiples fuentes de información y prepararlos para visualización o Machine Learning. Es la habilidad número uno que buscan los empleadores en Ciencia de Datos.

¿Para quién es este curso?

Excel Users

Para analistas que se sienten limitados por la capacidad de filas de Excel y quieren automatizar sus reportes.

Data Aspirants

Paso obligatorio para cualquier persona que desee convertirse en Científico de Datos o Analista de Datos profesional.

Contenido Programático

1. Estructuras de Datos

  • Introducción al ecosistema Pandas.
  • pd.Series: Arrays etiquetados.
  • pd.DataFrame: Tablas de datos.
  • Carga de datos (CSV, Excel, SQL, JSON).

2. Limpieza de Datos

  • Detección de valores nulos (NaN).
  • Imputación y eliminación: fillna(), dropna().
  • Eliminación de duplicados.
  • Conversión de tipos de datos (Casting).

3. Manipulación

  • Selección e Indexación (loc, iloc).
  • Filtrado condicional avanzado (Querying).
  • Creación de columnas calculadas.
  • Ordenamiento y ranking de valores.

4. Análisis Avanzado

  • Agrupaciones poderosas: groupby().
  • Tablas pivote (Pivot Tables) estilo Excel.
  • Fusión de datasets: merge(), concat().
  • Estadística descriptiva y correlaciones.
