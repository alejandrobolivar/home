Objetivo del Curso

Objetivo General: Comprender y aplicar el álgebra lineal computacional mediante la librería Numpy, base fundamental para la Ciencia de Datos y el Machine Learning.


NumPy (Numerical Python) es la librería que hace posible que Python sea rápido en cálculos matemáticos. En este curso dejarás de usar bucles lentos para pasar a la "vectorización", permitiéndote procesar millones de datos en fracciones de segundo, habilidad esencial para Pandas, Scikit-Learn y TensorFlow.

¿Para quién es este curso?

Requisitos

Conocimientos básicos de Python (variables, listas) y nociones elementales de matemáticas (vectores y matrices).

Resultado

Dominarás la creación de arrays, indexación avanzada y operaciones matriciales necesarias para el análisis de datos profesional.

Contenido Programático

1. Arrays vs Listas

  • Diferencias de rendimiento y memoria.
  • Creación de `ndarray`.
  • Tipos de datos (dtypes) en NumPy.
  • Arrays multidimensionales (Matrices y Tensores).

2. Operaciones Vectorizadas

  • Concepto de Vectorización (sin bucles for).
  • Broadcasting: Operar arrays de distintas formas.
  • Funciones Universales (ufuncs): sqrt, exp, log.
  • Operaciones aritméticas y lógicas eficientes.

3. Manipulación de Arrays

  • Indexación y Slicing (Rebanado) avanzado.
  • Indexación Booleana (Filtros/Máscaras).
  • Redimensionamiento (`reshape`, `flatten`).
  • Concatenación y división de arrays.

4. Álgebra Lineal

  • Producto punto y producto matricial (`@`).
  • Transpuesta e inversa de matrices.
  • Generación de números aleatorios (`numpy.random`).
  • Estadística básica (media, varianza, percentiles).
