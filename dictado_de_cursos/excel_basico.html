Objetivo del Curso

Objetivo General: Familiarizarse con el entorno de hojas de cálculo, realizando operaciones matemáticas básicas y organizando información de manera eficiente.


Excel es la herramienta número uno en el mundo empresarial. Este curso está diseñado para llevarte desde cero absoluto hasta sentirte cómodo navegando por la interfaz. Aprenderás a dejar de usar la calculadora manual y empezar a usar el poder de las fórmulas automáticas, además de presentar tus datos de forma clara y profesional.

¿Para quién es este curso?

Principiantes Absolutos

Personas que nunca han abierto Excel o que solo lo utilizan para hacer listas simples.

Administrativos

Quienes necesitan organizar inventarios, presupuestos básicos o bases de datos de clientes.

Contenido Programático

1. Introducción al Entorno

  • Conociendo la interfaz: Cinta de opciones y barras.
  • Conceptos clave: Libro, Hoja, Fila, Columna y Celda.
  • Navegación y selección eficiente de datos.
  • Tipos de datos (Texto vs. Número).

2. Fórmulas Básicas

  • La barra de fórmulas y el signo igual (=).
  • Operadores aritméticos (+, -, *, /).
  • Funciones esenciales: SUMA y PROMEDIO.
  • Encontrar valores: MAX y MIN.

3. Formato y Presentación

  • Fuentes, alineación, bordes y rellenos de color.
  • Formatos de número: Moneda ($), Porcentaje (%) y Millares.
  • Manejo correcto de Fechas y Horas.
  • Uso de Estilos de Celda predefinidos.

4. Gestión de Datos

  • Ordenar datos alfabética y numéricamente.
  • Uso de Filtros básicos (el embudo).
  • Buscar y Reemplazar información.
  • Configuración de página e Impresión correcta.
