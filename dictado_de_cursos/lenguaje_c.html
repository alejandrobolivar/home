Objetivo del Curso

Objetivo General: Aprender las bases de la programación de sistemas, entendiendo la gestión manual de la memoria y el funcionamiento a bajo nivel del computador.


El lenguaje C es conocido como el "padre" de los lenguajes modernos. En este curso no solo aprenderás sintaxis; aprenderás cómo funciona la máquina por dentro. Dominar C te dará una comprensión profunda sobre eficiencia, punteros y estructuras de datos que te convertirá en un programador superior en cualquier otro lenguaje.

¿Para quién es este curso?

Ing. Electrónica/Sistemas

Para quienes desean programar microcontroladores, sistemas embebidos o sistemas operativos.

Backend Performance

Para desarrolladores que necesitan optimizar código crítico donde cada byte de memoria cuenta.

Contenido Programático

1. Fundamentos

  • Estructura de un programa en C.
  • Compilación (GCC) y Enlazado.
  • Tipos de datos primitivos (`int`, `float`, `char`).
  • Operadores aritméticos y lógicos.

2. Control y Funciones

  • Sentencias `if`, `switch`, `while`, `for`.
  • Programación Modular (Header files `.h`).
  • Ámbito de variables (locales, globales, estáticas).
  • Paso de parámetros por valor.

3. Arreglos y Cadenas

  • Vectores (Arrays unidimensionales).
  • Matrices (Arrays multidimensionales).
  • C-Strings: Manipulación de texto como arrays de caracteres.
  • Librería `string.h`.

4. Punteros y Memoria

  • Direcciones de memoria (`&`) y Punteros (`*`).
  • Aritmética de Punteros.
  • Paso de parámetros por referencia.
  • Estructuras (`struct`): Datos complejos.
  • Gestión dinámica de memoria (`malloc`, `free`).
