Objetivo del Curso

Objetivo General: Construir aplicaciones web robustas y escalables utilizando el micro-framework Flask de Python, comprendiendo a profundidad el funcionamiento del Backend.


A diferencia de frameworks pesados, Flask te da el control total. En este curso no solo aprenderás a escribir código, sino a entender cómo funciona el servidor, cómo viajan los datos HTTP y cómo conectar tu aplicación con bases de datos reales. Es el paso definitivo para convertirte en un Backend Developer.

¿Para quién es este curso?

Backend Aspirants

Para desarrolladores Python que quieren saltar de la consola a la web, creando APIs y sitios dinámicos.

Full Stack Devs

Para quienes ya saben Frontend (HTML/CSS) y necesitan un motor lógico potente y flexible en el servidor.

Contenido Programático

1. Fundamentos

  • El patrón de diseño MVT (Modelo-Vista-Template).
  • Rutas y Vistas: @app.route('/').
  • Estructura profesional de carpetas (static, templates).
  • Manejo de variables en la URL.

2. Plantillas Jinja2

  • Renderizado de HTML dinámico: render_template.
  • Lógica en HTML: Bucles y Condicionales.
  • Herencia de plantillas: {% extends %}.
  • Filtros y macros personalizados.

3. Datos y Formularios

  • Métodos HTTP: GET vs POST.
  • Manejo de Formularios con WTForms.
  • Conexión a Bases de Datos (SQLAlchemy).
  • Validación de datos y mensajes flash.

4. Arquitectura

  • Autenticación de usuarios (Flask-Login).
  • Hash de contraseñas y seguridad.
  • Escalabilidad con Blueprints.
  • Despliegue básico en servidor (Deploy).
