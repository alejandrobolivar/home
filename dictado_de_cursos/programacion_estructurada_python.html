Descripción y Objetivos

Objetivo General: El participante reconocerá los elementos básicos de programación, identificará la sintaxis y estilo de programación con el lenguaje Python, que le permitirán crear scripts y programas en este lenguaje de alto nivel, en entornos de escritorio, para procesar y extraer datos.


Python es un lenguaje interpretado con una sintaxis específica que fomenta el código limpio y fácil de mantener. Este curso introduce el paradigma, manejo de variables, manipulación de archivos y entrada/salida de datos, permitiendo crear programas sencillos como calculadoras o traductores, sentando las bases para lenguajes como C, C++ o Java.

Temario del Curso

1. El Lenguaje Python

  • Descripción general y características.
  • Primer programa en Python.
  • Variables y Tipos: Valores, palabras clave, sentencias.
  • Expresiones: Operadores, operandos y evaluación.
  • Operaciones con cadenas y comentarios.

2. Control de Flujo

  • Estructuras Condicionales: Operadores lógicos.
  • Ejecución condicional, alternativa y encadenada.
  • Estructuras Repetitivas:
  • La sentencia for.
  • La sentencia while.

3. Datos Estructurados

  • Listas: Creación y manipulación.
  • Tuplas: Características y usos.
  • Diccionarios: Clave-valor.
  • Conjuntos (Sets).

Perfiles y Requisitos

Dirigido a

Estudiantes de áreas de ingeniería, informática y público en general con nociones básicas que buscan refinar sus habilidades y aprender un nuevo lenguaje.

Perfil de Ingreso

Programadores, estudiantes de áreas afines, así como personas con fundamentos o nociones de cualquier lenguaje de programación.

Perfil de Egreso

Al finalizar, sabrás utilizar la sintaxis de Python para aplicar el procesamiento de datos afines a tu carrera, usando este lenguaje como herramienta principal.
