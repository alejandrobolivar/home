Objetivo del Curso

Objetivo General: Desarrollar aplicaciones de escritorio completas con ventanas, botones y menús utilizando librerías nativas profesionales como PyQt (o Tkinter).


Python es poderoso, pero tus usuarios finales necesitan interfaces visuales amigables. En este curso aprenderás a usar Qt Designer para diseñar ventanas arrastrando elementos y luego darles vida con código Python, terminando con la creación de un archivo ejecutable .exe que podrás instalar en cualquier computadora con Windows.

¿Para quién es este curso?

De Script a App

Para programadores que ya saben Python básico y quieren transformar sus scripts de consola en aplicaciones visuales distribuibles.

Desarrollo de Tools

Ideal para crear herramientas internas de empresa, sistemas de gestión o paneles de control que requieren interacción visual.

Contenido Programático

1. Ventanas y Widgets

  • Introducción a PyQt / Tkinter.
  • Ventana Principal (MainWindow) y Diálogos.
  • Widgets Básicos: Labels, Botones, Inputs de texto.
  • Estilización con CSS (QSS).

2. Layouts (Diseño)

  • Organización geométrica de elementos.
  • Grid Layout, Vertical & Horizontal Layouts.
  • Diseño responsivo (redimensionable).
  • Uso de Qt Designer (Drag & Drop).

3. Interactividad

  • Manejo de Eventos (Clics, Teclado).
  • Signals & Slots: Conectando la lógica visual.
  • Validación de datos de entrada.
  • Ventanas emergentes (Popups y Alertas).

4. Distribución (.exe)

  • Empaquetado de aplicaciones.
  • Uso de PyInstaller.
  • Inclusión de imágenes y recursos.
  • Generación del instalador o archivo .exe final.
