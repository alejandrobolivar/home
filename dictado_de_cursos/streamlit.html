Objetivo del Curso

Objetivo General: Aprender a convertir scripts de análisis de datos en aplicaciones web interactivas de manera rápida y eficiente, sin necesidad de conocimientos profundos de Frontend.


¿Eres científico de datos o analista y quieres compartir tus modelos? Olvídate de aprender Django, Flask, HTML o React. Con Streamlit, todo es Python. En este curso aprenderás a crear dashboards, herramientas de IA y reportes interactivos en cuestión de horas, no semanas.

¿Para quién es este curso?

Data Scientists

Para quienes necesitan mostrar sus modelos y gráficas a clientes sin enviar capturas de pantalla estáticas.

Python Devs

Para programadores que desean prototipar interfaces de usuario (UI) a la velocidad del rayo.

Contenido Programático

1. Introducción

  • ¿Por qué Streamlit? Filosofía y ventajas.
  • Instalación y estructura: pip install streamlit.
  • El comando mágico: st.write().
  • Flujo de ejecución y "Hot Reload".

2. Widgets Interactivos

  • Entrada de datos: st.text_input.
  • Controles numéricos: st.slider.
  • Botones y Acciones: st.button.
  • Selección: st.selectbox, st.multiselect.

3. Visualización Web

  • Integración nativa con Pandas DataFrames.
  • Gráficos simples: st.line_chart.
  • Integración con Matplotlib, Plotly y Altair.
  • Mostrar métricas e indicadores clave (KPIs).

4. Despliegue

  • Diseño y Layouts: st.sidebar, st.columns.
  • Gestión del estado de sesión (Session State).
  • Configuración del tema (Dark/Light Mode).
  • Publicación gratis en Streamlit Cloud.
